## Link

A link represents a link on the list.

| Field                                  | Type     | Description           |
| -------------------------------------- | -------- | --------------------  |
| name                                   | `String` | The name of the link  |
| value                                  | `String` | The value of the link |

**Example**

```json
{"name": "Name of the link", "value": "http://example.com"}
```
