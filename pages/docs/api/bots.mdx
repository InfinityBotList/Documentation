## Post Bot Stats

<HTTPHeader type="POST" path="bots/stats" />

### Post Headers

| Field         | Type     | Description                        | Required |
| ------------- | -------- | ---------------------------------- | -------- |
| Authorization | `string` | The Bots `Infinity Bots API Token` | ‚úîÔ∏è       |

### Post Body

| Field      | Type       | Description                      | Required |
| ---------- | ---------- | -------------------------------- | -------- |
| servers    | `uint64`   | Amount of servers the bot is in. | ‚úîÔ∏è        |
| shards     | `uint64`   | Amount of shards the bot has.    | ‚ùå       |
| users      | `uint64`   | Amount of users the bot has.     | ‚ùå       |
| shard_list | `[]uint64` | Shard list of bot.               | ‚ùå       |

```py
import requests
req = requests.post(f"{API_URL}/bots/stats", json={"servers": 4000, "shards": 2}, headers={"Authorization": f"{TOKEN}"})
print(req.json())
```
